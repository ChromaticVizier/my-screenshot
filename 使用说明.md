# My Screenshot 使用说明

一个浏览器截图 / 录屏插件。

## 安装

Clone 仓库：https://gitlab.com/zjq8947/my-screenshot.git，需 Node v24.10.0。

```bash
pnpm i
pnpm build
```

Chrome → 扩展程序 → 开发者模式 → 加载已解压的扩展程序 → 选 `build/chrome-mv3-prod`。

*或* 直接解压压缩包使用。

## 截图

点击工具栏图标，打开 popup，切到「截图」面板。

| 功能 | 作用 |
|---|---|
| 可视区域 | 截当前视口。 |
| 整个页面 | 长截图，截完整页。 |
| 选择区域 | 鼠标拖拽框选要截的区域。 |
| 延迟截取可视区域 | 倒计时后再截视口，方便先调整页面。秒数在设置里改。 |
| 整个屏幕或应用窗口 | 从系统选择器里选屏幕、窗口或某个标签页截图。 |
| 选择滚动区域 | 手动指定长截图的滚动容器，按网站记忆，下次访问同站自动复用。 |
| 清除本网站区域 | 清掉当前网站记住的滚动容器。 |
| 标注本地或剪贴板图片 / 截屏提取文本 | 暂未实现。 |

底部齿轮图标进入设置页；「云端」按钮暂未实现。

## 录屏

切到「录屏」面板，选录制模式 → 调录制选项 → 点「开始录制」。再次点击同一按钮结束录制并下载。

录制模式：

- **当前标签页**：录当前打开的网页。可暂停 / 继续 / 结束。
- **区域录制（当前标签页）**：先在页面上框选区域再录。
- **桌面 / 摄像头**：暂未接入。

录制选项（按钮在模式卡片下方）：

- **麦克风**：开关麦克风。
- **系统声音**：开关网页 / 系统内部声音。
- **控制栏**：开关页面上的悬浮录制控制条（功能待实现）。
- **分辨率**：720p / 1080p / 4K。
- **格式**：WebM（MP4 暂禁用）。
- **云端存储**：暂未实现。

录制中关闭 popup 不会中断录制；重新打开 popup 仍能看到「结束录制」按钮。

## 设置

popup 右下角齿轮进入。

- **延迟截图秒数**：延迟截图的等待时间，1~60 秒。
- **整页截图判别规则**：约 20 项阈值，控制长截图时哪些吸顶 / 浮层 / 侧栏元素被隐藏，哪些保留。每项右侧有「恢复默认」；页面顶部可整体重置 / 导出 / 导入 JSON。
- 若长截图出现「该隐藏的元素重复出现」或「该保留的内容被误隐藏」，可针对性调对应阈值，或在「自定义保留 / 隐藏选择器」里直接填 CSS selector 强制处理。

## 已知限制

视频方面：

- MP4 输出不可用，webm编解码并不是100%稳定，如遇视频无法播放可尝试重新录制。
- 摄像头 / 桌面录屏未接通。
- 系统声音录制未充分验证。
- 标注、OCR、云端上传未实现。

长截图方面：

- 对于流式渲染/虚拟滚动的聊天框（宝库、deepseek聊天页面等）截图效果不佳，可能产生空白或元素遮挡。
- 涉及动态加载、有dom回收的页面（mooc等）会出现下方空白。
